# Vue 2 Blog Project

Project made following The Net Ninja Vue JS 2 Tutorial.

## Technologies

- Vue 2
- VueRouter
- JSON Server

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
json-server --watch ./data/db.json
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
